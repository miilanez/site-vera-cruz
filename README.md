## Vamos Começar

Primeiro, clone o repositório

```
$ git remote add origin https://github.com/miilanez/site-vera-cruz.git
```

Depois, carregue as dependências

```
$ cd site-vera-cruz
$ npm install
``` 
Por último, mas não menos importante, execute o servidor

```
$ npm start
```
